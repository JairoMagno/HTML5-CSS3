Principais formatações

Negrito / Destaque

Para usar algum termo em negrito, basta usar a tag B (não semântica)

Para usar algum termo forte/destaque, basta usar a tag STRONG (semântica)

Itálico / Ênfase

Para usar algum termo em itálico, basta usar a tag I (não semântica)

Para usar algum termo em ênfase, basta usar a tag EM (semântica)

Texto marcado

É possível marcar um texto, com a tag MARK

A cor padrão da tag MARK é o amaralo, porém é possível usar outra cor padrão na seção <head>

Texto grande e pequeno

Posso criar um Texto Grande com a tag BIG (está obsoleta), e um Texto Pequeno com a tag SMALL

Texto deletado

podemos marcar um texto excluído com a tag DEL

Texto inserido

Podemos marcar um texto inserido com a tag INS

também podemos sublinhar da mesma forma com a tag U (Apenas forma, sem semântica)

Texto sobrescrito

Podemos sobrescrever o texto: x2 + 3. Para isso basta usar a tag SUP

Texto subscrito

Podemos subscrever o texto: H2O. Para isso basta usar a tag SUB
